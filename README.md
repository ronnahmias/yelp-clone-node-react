# Yelp Clone - In progress

yelp clone app using PERN stack:
backend: nodejs
frontend: react
sql DB: postgres

## Requirements

[Node](https://nodejs.org/en/) Installation

## Installation

clone repo and open directory

install dependencies

```bash
cd server
npm install
```

## Usage
init server
```bash
npm run start
```
